_Genesis entry — first run of BO-env-documenter. All currently-documented vars are listed as Added._

### Added
- `LOG_LEVEL` — used in `backend/config.py`. Optional, default `INFO`. Type: string. Application log verbosity.
- `VITE_API_BASE_URL` — used in `src/components/workspace/SubmitCaseDialog.tsx`, `src/components/workspace/DocumentViewer.tsx`, `src/lib/adminApi.ts`, `src/contexts/AppContext.tsx`. Optional, default `http://localhost:8000`. Type: URL. Frontend → backend API base URL.
- `VITE_API_URL` — used in `src/components/workspace/AdminConfigPanel.tsx`, `src/components/workspace/AIChatInterface.tsx`, `src/components/workspace/CaseContextDialog.tsx`, `src/components/workspace/CaseTreeExplorer.tsx`, `src/components/workspace/ContextHierarchyDialog.tsx`, `src/lib/fileApi.ts`. Optional, default `http://localhost:8000`. Type: URL. Frontend → backend API base URL.
- `DOCUMENTS_PATH` — used in `backend/config.py::get_documents_path`. Optional, default `public/documents`. Type: filesystem path. Document storage base path.
- `GEMINI_API_KEY` (sensitive) — used in `backend/services/gemini_service.py::_initialize_model`, `backend/services/validation_service.py::_initialize_model`, `backend/services/shacl_generator.py`. Required (raises `ValueError` if missing). Type: string. Google Gemini API key.
- `IDIRS_BASE_URL` — used in `backend/config.py`. Optional, default `http://localhost:8010`. Type: URL. IDIRS hybrid-search/RAG service base URL.
- `INIT_TEST_DOCS` — used in `backend/main.py`. Optional, default `false`. Type: boolean. Auto-initialise test documents on startup.
- `SKIP_INTEGRATION_TESTS` — used in `backend/tests/test_anonymization.py`. Optional, default `1`. Type: boolean. Skip anonymization integration tests.
